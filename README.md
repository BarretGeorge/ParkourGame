# Unity 6 游戏项目

这是一个使用 Unity 6 创建的游戏项目。

## 项目结构

```
Assets/
├── Scripts/     # C# 脚本文件
├── Scenes/      # 游戏场景
├── Art/         # 美术资源（模型、材质等）
└── Prefabs/     # 预制件

ProjectSettings/ # Unity 项目设置
Packages/        # 包管理配置
UserSettings/    # 用户特定设置（不提交到版本控制）
```

## 开始开发

1. 使用 Unity 6 打开此项目
2. 打开 `Assets/Scenes/MainScene.unity` 场景
3. 在 `Assets/Scripts/` 中创建你的脚本
4. 开始游戏开发！

## 已包含的组件

- `GameManager.cs` - 游戏主管理器，包含基本的游戏流程控制
- `MainScene.unity` - 初始场景，包含主摄像机和光源

## 版本控制

此项目已配置 `.gitignore`，请使用 Git 进行版本控制。
